# Dvorak

Create Dvorak decks using YAML files. Generates PDFs of cards and rules.

## Installation

Install it with:

    $ gem install dvorak

## Usage

    $ dvorak new mad_world
    $ cd mad_world

Edit the files in your project directory, then do:

    $ dvorak generate # Well, eventually...
